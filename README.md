## Установка

```bash
cd StreamSwitcher
pip install -r requirements.txt
```

## Запуск

```bash

```

## Как всё работает

### 1. Источники звука (Source Switching)

В верхней панели три кнопки:
- **🎤 Живой вход** — звук с микрофона/Line-In через выбранный аудиоинтерфейс
- **🎵 MP3 файл** — воспроизведение из плейлиста (локальные файлы, HTTP, SMB/UNC пути)
- **📻 Радио** — интернет-радио поток (Icecast/Shoutcast/HTTP)

Переключение между источниками происходит **плавно** (Fade-out 1 сек → Fade-in 1 сек).

### 2. Dual Mix Mode (НОВОЕ)

Кнопка **🔀 Dual Mix** включает одновременное воспроизведение двух источников:
- **Live** (микрофон) + **Secondary** (MP3 или радио)
- Независимые слайдеры громкости для каждого в левой панели
- Выбор вторичного источника через выпадающий список

Пример: ведущий говорит в микрофон, а фоном играет музыка из плейлиста.

### 3. Auto-Failover (НОВОЕ)

Если включён чекбокс **"Auto-Failover при обрыве"**:
- Система мониторит текущий источник каждые 2 секунды
- Если источник молчит **>8 секунд** — автоматическое переключение на следующий по цепочке:
  `Live → MP3 → Radio → Live → ...`

Дополнительно: **Silence Detector** — если живой вход молчит >30 сек, запускается резервный плейлист.

### 4. Транспорт

| Кнопка | Действие |
|--------|----------|
| ▶ Play | Запуск текущего источника |
| ⏹ Stop | Остановка воспроизведения |
| ⏭ Next | Следующий трек в плейлисте |
| 🔇 Mute | Заглушить выход |

### 5. Audio Matrix (левая панель)

- **Вход** — выбор входного устройства (WASAPI/DirectSound/MME)
- **Выход** — выбор выходного устройства
- **Sample Rate** — 44100 / 48000 / 88200 / 96000 Hz
- **VU Meters** — индикаторы уровня с пиковыми метками и детектором клиппинга

### 6. Плейлист (вкладка 🎵)

- **Добавить файлы** — локальные MP3/WAV/FLAC/OGG
- **Добавить URL** — HTTP/FTP ссылка на файл
- **SMB/NFS путь** — сетевое хранилище (`\\server\share\file.mp3`)
- Двойной клик — воспроизвести трек
- Drag & Drop для перестановки

### 7. Радио (вкладка 📻)

- Ввод URL потока и кнопка "Слушать"
- Готовые пресеты (двойной клик для запуска)
- Индикатор буферизации

### 8. Расписание (вкладка ⏰)

Планировщик для автоматических действий по времени:

- Задать **HH:MM:SS** → выбрать действие → указать цель → добавить
- Действия: воспроизвести файл, включить радио, переключить на вход, остановить
- **Редактирование** — двойной клик по записи или кнопка "✏ Редактировать"
- Вкл/Выкл отдельных записей
- Ежедневный повтор или одноразовое срабатывание

### 9. DSP (вкладка 🎛)

- **5-полосный эквалайзер** (60Hz, 250Hz, 1kHz, 4kHz, 12kHz) ±12 dB
- **Компрессор/Лимитер** — порог, соотношение, усиление
- Включаются независимо чекбоксами

### 10. Стриминг (вкладка 📡)

Вещание на Icecast2/Shoutcast:
- Настройки: хост, порт, маунт, пароль, битрейт
- Кнопка "Начать вещание" / "Остановить"
- Отображение: статус, количество слушателей, отправлено данных
- **Auto-Reconnect** при разрыве соединения

### 11. Remote API

Встроенный веб-сервер на порту **8080**:
- Откройте `http://<IP>:8080` на смартфоне
- Кнопки: Play, Stop, Next, Mute
- Переключение источников
- Отображение статуса (источник, аптайм, слушатели, трек)

### 12. Waveform Mini-Player

Внизу окна — визуализация волны текущего MP3:
- Показывает прогресс воспроизведения
- Клик по волне = перемотка

## Структура проекта

```
StreamSwitcher/
├── main.py              # Точка входа
├── requirements.txt     # Зависимости
├── core/
│   ├── audio_engine.py  # Ядро: захват, микс, DSP, вывод
│   ├── source_manager.py # MP3/Radio декодирование
│   ├── scheduler.py     # Планировщик событий
│   ├── streamer.py      # Icecast вещание
│   └── remote_api.py    # Flask REST API + веб-пульт
└── ui/
    ├── main_window.py   # Главное окно
    ├── styles.py        # Dark Mode стили
    ├── vu_meter.py      # VU-метры
    ├── waveform_widget.py # Визуализация волны
    ├── dsp_panel.py     # EQ + Компрессор
    ├── scheduler_panel.py # UI расписания
    └── stream_panel.py  # UI стриминга
```

- Python 3.10+
- Windows (WASAPI) / macOS / Linux (ALSA/PulseAudio)
- PySide6, sounddevice, soundfile, numpy, flask, scipy, requests
